fmtPlayer
directive in module fmt

Description

HTML element that shows a player.

Usage

as element:
<fmt-player
       content="{string}"
       [src="{string}"]
       [player-html5="{string}"]
       [player-flash="{string}"]
       [tech-order="{string}"]
       [poster-url="{string}"]
       [autoplay-enabled="{boolean}"]
       [seo-enabled="{boolean}"]
       [stats-enabled="{boolean}"]>
</fmt-player>
as attribute
<ANY fmt-player
     content="{string}"
     [src="{string}"]
     [player-html5="{string}"]
     [player-flash="{string}"]
     [tech-order="{string}"]
     [poster-url="{string}"]
     [autoplay-enabled="{boolean}"]
     [seo-enabled="{boolean}"]
     [stats-enabled="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
contentstring

Flumotion ID of the video / audio to reproduce.

src
(optional)
string

Specify a stream URL to force it instead of using an ID.

playerHtml5
(optional)
string

Select the player to use for HTML5 technology. Options are browser and videojs.

playerFlash
(optional)
string

Select the player to use for Flash technology. Options are strobe, strobe16 and videojs.

techOrder
(optional)
string

Defines the order of technologies using the first supported by the agent. By default it is html5_flash_dashjs.

posterUrl
(optional)
string

URL of the image to use as a video poster.

autoplayEnabled
(optional)
boolean

Enables the auto-play if true.

seoEnabled
(optional)
boolean

If set to true it will automatically add the current pod information to the URL using hashbang.

statsEnabled
(optional)
boolean

If set to true it will send statistics to Flumotion and Google Analytics.
