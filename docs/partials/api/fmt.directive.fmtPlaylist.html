fmtPlaylist
directive in module fmt

Description

HTML element that shows a list of items and synchronize them with the player directive.

Usage

as attribute
<ANY fmt-playlist
     content="{string}"
     [name-visible="{boolean}"]
     [thumb-visible="{boolean}"]
     [duration-visible="{boolean}"]
     [likes-visible="{boolean}"]
     [date-visible="{boolean}"]
     [current-item-id="{number}"]
     [auto-advance-enabled="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
contentstring

Specify the Flumotion list ID or JSON content URL to load items from.

nameVisible
(optional)
boolean

Shows items name. Visible by default.

thumbVisible
(optional)
boolean

Shows items thumbnail. Visible by default.

durationVisible
(optional)
boolean

Shows items duration. It requires thumbnail to be visible. Visible by default.

likesVisible
(optional)
boolean

Shows items likes. Visible by default.

dateVisible
(optional)
boolean

Shows items creation date. Hidden by default.

currentItemId
(optional)
number

ID of the item selected by default. If this is not specified the playlist will select the first item of the list. Use none to skip selecting an item.

autoAdvanceEnabled
(optional)
boolean

Enables the auto-advance when the directive receives an event fmtPodFinished Disabled by default.
