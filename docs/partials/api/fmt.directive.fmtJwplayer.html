fmtJwplayer
directive in module fmt

Description

Includes a configurable JW Player compatible with Flumotion OVRP.

CSS classes

  • fmt-jwplayer is set when the directive has been parsed.

Usage

as element:
<fmt-jwplayer
       content="{string}"
       [player-id="{number}"]>
</fmt-jwplayer>
as attribute
<ANY fmt-jwplayer
     content="{string}"
     [player-id="{number}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
contentstring

Source URL or Flumotion ID of the video / audio to reproduce.

playerId
(optional)
number

ID of a Flumotion player
