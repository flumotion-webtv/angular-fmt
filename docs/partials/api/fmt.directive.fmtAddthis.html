fmtAddthis
directive in module fmt

Description

AddThis toolbox wrapper that automatically gets synchronized with the pod selected by other Flumotion widgets like playlist directive.

Usage

as attribute
<ANY fmt-addthis
     [pubid="{string}"]
     [size="{string}"]
     [use-preferred="{boolean}"]
     [num-preferred="{number}"]
     [buttons="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
pubid
(optional)
string

AddThis ID used to manage the services. By default it uses one owned by Flumotion.

size
(optional)
string

Size of the buttons. Options are large and small. The default value is large.

usePreferred
(optional)
boolean

If true it will use the preferred buttons selected by AddThis. The default value is true. More info in Preferred versus Specified Services.

numPreferred
(optional)
number

Number of default buttons to show in case usePreferred is true. The default value is 5.

buttons
(optional)
string

Comma-separated string containing the buttons to be shown. The default value is facebook,google_plusone_share,twitter,linkedin,tuenti. More info in AddThis service directory.
